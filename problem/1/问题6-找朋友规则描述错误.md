# 问题6：找朋友规则描述错误

## 问题描述
多处文档将找朋友规则错误地描述为"第一个打出这张牌的玩家成为盟友"，但正确的规则应该是"庄家指定第N张，则第N个打出的玩家成为盟友"。

## 错误位置

### 1. templates/rules.html:149
```html
<strong>找朋友（核心）：</strong>庄家叫一张指定牌（花色+点数，例如：红桃A）。
游戏中第一个打出这张牌的玩家，自动成为庄家的秘密盟友。
```

**应改为**：
```html
<strong>找朋友（核心）：</strong>庄家叫一张指定牌（花色+点数+第几张，例如：第2张红桃A）。
游戏中第N个打出这张牌的玩家（N为庄家指定的张数），自动成为庄家的秘密盟友。
```

### 2. test/advanced_rules.go:418
```go
ts.AddResult("朋友揭示", true, "第一个打出叫牌的玩家成为盟友", nil)
```

**应改为**：
```go
ts.AddResult("朋友揭示", true, "第N个打出叫牌的玩家成为盟友（N为庄家指定）", nil)
```

### 3. test/测试报告.md:63
```markdown
- 第一个打出该牌的玩家成为盟友
```

**应改为**：
```markdown
- 第N个打出该牌的玩家成为盟友（N为庄家指定的张数）
```

## 正确规则说明

### 庄家叫牌时需要指定三个要素：
1. **花色**：黑桃/红桃/梅花/方片/大王/小王
2. **点数**：2-10/J/Q/K/A（若是王则不需要点数）
3. **第几张**：第1张、第2张、第3张等

### 示例：
- 庄家叫"第2张红桃A"：游戏中第2个打出红桃A的玩家成为盟友
- 庄家叫"第1张大王"：游戏中第1个打出大王的玩家成为盟友
- 庄家叫"第3张方片K"：游戏中第3个打出方片K的玩家成为盟友

### 特殊情况：
- 如果庄家叫的牌在自己手里，或者扣在底牌中，则无人应叫，庄家1打4
- 如果叫"第2张"但整场游戏只出现1张该牌，则无盟友

